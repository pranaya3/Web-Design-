How to Build a Simple Skincare Routine

Welcome to this interactive learning module! This lesson teaches beginners how to create an effective daily skincare routine using six essential products. By completing this lesson, learners will understand the purpose of each skincare product and apply them in the correct order.

Student Learning Objectives

  • Identify six common skincare products.
  • Explain the purpose of each product.
  • Apply products in the correct order.
  • Create a beginner skincare routine independently.

Step-by-Step Routine

1. Cleanser

Removes dirt, oil, and makeup.

2. Toner

Hydrates and prepares the skin.

3. Serum

Targets skin concerns with active ingredients.

4. Moisturizer

Locks in hydration.

5. Sunscreen

Protects skin from harmful UV rays.

Digital Media Principles

Images

The infographic uses icons, consistent colors, and organized sections to reduce cognitive load and improve understanding.

Video

The instructional video combines narration with visuals, allowing learners to observe each skincare step while hearing explanations.

Audio

The audio recording reinforces important concepts for auditory learners and provides a convenient review option.

Consistency

A consistent color palette, typography, and layout create a cohesive learning experience.

How the Digital Media Support Learning

The infographic introduces the skincare routine visually. The instructional video demonstrates how each product is used, while the audio summary reinforces the lesson through narration. Together, these multimedia components address different learning preferences and improve student understanding of the correct skincare sequence.
