Welcome to this interactive learning module! This lesson teaches beginners how to create a simple skincare routine using six essential products. Learners will understand the purpose of each product and the correct order of application.

🎯 Student Learning Objectives

  • Identify six essential skincare products.
  • Explain the purpose of each product.
  • Apply products in the correct order.
  • Create a beginner skincare routine.

🧴 Six Steps to a Simple Skincare Routine

1. Cleanser 🫧

CeraVe Foaming Facial Cleanser removes dirt and oil.

2. Toner 💧

Byoma Hydrating Milky Toner prepares and hydrates skin.

3. Serum ✨

Glow Recipe Niacinamide Dew Drops target skin concerns.

4. Moisturizer 🧴

COSRX Advanced Snail Cream locks in hydration.

5. Sunscreen ☀️

Supergoop SPF protects skin from UV rays.

6. Lip Treatment 💋

OLEHENRIKSEN Pout Preserve hydrates and protects lips.
