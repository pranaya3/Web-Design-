Beginner's Guide to Healthy Skin ✨

Welcome to this interactive skincare learning module! This lesson teaches beginners how to create a simple daily skincare routine using six essential products. Learners will understand what each product does and how the steps work together to support healthy skin.

🎯 Student Learning Objectives

  • Identify the six essential products in a beginner skincare routine.
  • Explain the purpose of each skincare product.
  • Apply skincare products in the correct order.
  • Create a personalized skincare routine.

💻 Digital Media Principles

Images

The infographic uses visuals, icons, and organization to reduce cognitive load and help learners understand information quickly.

Video

The video combines narration and visual demonstrations to help learners see how each skincare step should be completed.

Audio

The audio summary supports auditory learners by allowing them to review the lesson through narration.

Consistency

The consistent colors, typography, and layout create an engaging and organized learning experience.

🌱 Reflection Activity

After completing this lesson, reflect on the following:

1. Which skincare product do you think is the most important and why?
2. How would you explain a skincare routine to someone who has never created one?

How the Digital Media Support Learning

The infographic introduces the skincare routine visually. The instructional video demonstrates each step, while the audio summary reinforces important concepts. Together, these multimedia elements support different learning styles and help learners understand the correct order and purpose of each skincare product.
